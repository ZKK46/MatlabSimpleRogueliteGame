1.运行“my_game.m”，这是游戏游玩的主程序。

2.主菜单显示的四个选项分别是
1）开始
弹出新窗口后会让玩家选择皮肤：即颜色、形状和技能（起初只有默认的），选择完毕后游戏开始。
游戏开始后玩家可以通过键盘上的上下左右键控制像素块移动方向（当玩家未操作时会保持上一次移动的方向一直移动）。当像素块接触到boss时，玩家可以通过按下空格键来攻击boss。此外，屏幕上还会有一颗子弹（一个和boss颜色一样的“*”），如果它接触到像素块，就记为攻击玩家一次。
如果boss的hp=0并且玩家的hp>0，玩家获胜。胜利将获得经验值（在商店购买新的皮肤、技能）和boss的颜色（如果玩家已经获得了boss的这一色彩，玩家将获得额外的经验值）；如果boss的hp＞0，而玩家的hp＝0，玩家失败。

2）商店
玩家可以在商店里花3经验值购买新的皮肤。
值得一提的是，为了增加趣味性，皮肤商店有一个“随机”选项：如果选择它，玩家将花费2经验值来随机获得一个形状。但如果玩家获得的是一个玩家背包里已经拥有的皮肤，那么玩家会白白损失2经验值。

3）背包
玩家可以在背包里随时查看玩家拥有的经验值、颜色、形状和技能。

4）设置
玩家可以在设置里改变游戏的难度（即子弹的移动速度和攻击力）。boss的困难越大，玩家获得胜利时就能得到越多的分数。
